数据库部分：
localhost 中，选择或者新建mysql数据库。点击操作会看到在数据库 mysql 中新建一张数据表。名字中写floor，点击执行会弹出一个新建数据建库的界面。
在字段中填写floor/class ，类型选择为int,界面下拉，在底部可以看到添加1字符 执行可以添加字符，自行根据字符数设定，我们一共需要设定9个字符。
第一个地段已经填写好，接下8个我们填写数字1~8 ，类型选择为text。随后点击保存，点击 floor 进入 。
接下来点击插入 在floor/class字段的值中填写1，字段数字1~8的值全部填写ON，点击执行。
跳转到SQL页面，此时我们已经将填写的东西插入好了，在页面中可以看到（在数据库mtsql运行SQL查询），我们把其中values后的1修改分别改成2、3、4、5然后点击执行。数据表的建立就完成了


功能：
Api：mapApi使用了FastApi来搭建，功能是获取玩家在mc中的坐标，并可以给玩家设定坐标。

前端：frontend使用了React Hook 搭建，并使用了canvas画出交互界面，界面实时显示玩家在地图中的位置，
并可通过鼠标点击设定玩家坐标。背景地图在CSS中设定。

环境要求：
1.安装FastApi环境，命令：pip install fastapi
		         pip install uvicorn
2.安装完成，在mapApi目录中运行：uvicorn main:app --reload
(可在main.py中修改mc服务器地址和端口)

3.安装好yarn 包管理工具

4.安装好node.js

5.安装Ant-Design组件库

6.cd到frontend文件夹，运行：yarn start
